# Lógica de Programação - DIO

Este repositório contém os desafios e exercícios realizados durante o curso de **Lógica de Programação** da [Digital Innovation One (DIO)](https://web.dio.me/track/formacao-logica-de-programacao).

## 🚀 Sobre o curso
O curso teve como base o desenvolvimento em **JavaScript**, abordando conceitos fundamentais de lógica de programação e resolução de problemas.  
Durante a formação, foram trabalhados:
- Estruturas de controle (condicionais e loops)
- Funções e escopo
- Arrays e objetos
- Classes e orientação a objetos
- Manipulação de strings e números
- Exercícios práticos e desafios de código

## 📂 Conteúdo do repositório
- **Desafios práticos:** exercícios propostos pela DIO para aplicar os conceitos aprendidos.
- **Exemplos de código:** soluções desenvolvidas em JavaScript.
- **Projetos de treino:** pequenos programas para fixar lógica e sintaxe.

## 🛠️ Tecnologias utilizadas
- **JavaScript (ES6+)**
- **Node.js** para execução dos códigos no terminal
- **Git & GitHub** para versionamento e hospedagem

## 🎯 Objetivo
Consolidar a base de lógica de programação e aplicar os conceitos em projetos futuros de desenvolvimento web e software.
